Pastebin fields
These are the fields that are available and the meanings.
Paste Name
This an optional field which becomes the title of your Pastebin entry.
Paste Code
This is the only required field enter the text you want to share here.
To highlight particular lines, prefix each line with @h@
Advanced Settings
Toggle this switch to view and change the advanced settings.
These are all optional settings.
Paste Format
Select the programming language to enable syntax highlighting.
Paste Expiration
How long you want your post to remain in the Pastbin.com database.
Paste Exposure
Do you want your post to be public or private?
Remember these settings
Tick this and the advanced settings will be stored in the app.
